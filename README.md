# HERE SDK for Flutter - API Reference

This folder contains the offline version of the API Reference for this release. You can always find the latest version online at [HERE SDK for Flutter Documentation](https://www.here.com/docs/category/here-sdk-flutter).

The API Reference has been generated with Dartdoc. To view it, open `index.html` in your browser. For the best reading experience, follow the Dartdoc [guidelines](https://dart.dev/tools/dart-doc#view-local) on how to view the API Reference locally using a local web server.
